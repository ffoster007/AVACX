# Contributing to AVACX

Thanks for wanting to contribute. This file is intentionally short.

Setup and detailed developer instructions are in [DEVELOPERS.md](DEVELOPERS.md)

Quick reference

- Install dependencies: `npm install`
- Dev server: `npm run dev`
- Lint: `npm run lint`
- Build: `npm run build`
- Tests: `npm test`

Pull request checklist

- [ ] Lint, build, and tests pass locally
- [ ] Add or update tests for changes
- [ ] Update docs if behavior or setup changed
- [ ] Use clear commit messages (e.g. `feat:`, `fix:`, `docs:`)

Issues & security

- Search existing issues before opening a new one.
- For security reports, see `SECURITY.md`.

Thank you — we appreciate your contributions!
