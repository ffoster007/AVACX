Getting Started
Fork and clone this repo.
```
git clone https://github.com/your-github-account/AVACX.git
```

## Quick Start
Install Node.js (v16 or higher) from [nodejs.org](https://nodejs.org/).
After that, start installing dependencies for the node.
```
npm install |or| npm i --force
```
Start the development server.
```
npm run dev
```
